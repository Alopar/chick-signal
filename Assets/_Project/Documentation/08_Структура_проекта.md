# 08. Структура проекта

Актуальная карта папок после уборки.

## Корень репозитория

```
ludum-dare-template/
  Assets/                 весь контент Unity-проекта
  Packages/               manifest + lock
  ProjectSettings/        Unity ProjectSettings (GraphicsSettings, TagManager, и т.д.)
  Library/ Temp/ Logs/    временные кэши Unity (в .gitignore)
  README-SETUP.md         краткий английский setup + ссылка на Documentation
```

## Assets (только папки, никаких «бродячих» файлов в корне)

```
Assets/
  _Project/               весь код и контент проекта
  Plugins/                Demigiant/DOTween (если поставлен) и другие сторонние SDK
  Settings/               UniversalRP · Renderer2D · UniversalRenderPipelineGlobalSettings · DefaultVolumeProfile
  TextMesh Pro/           TMP Essentials (после импорта из меню)
```

**Правило:** в `Assets/` на верхнем уровне живут **только папки**. Любой отдельный ассет (профиль, preset, asmdef) должен быть в подпапке.

## Assets/_Project

```
_Project/
  Art/                    Sprites · UI · Fonts · Animations · VFX
  Audio/                  Music · SFX · Mixers/MainMixer.mixer
  Documentation/          эта папка (русская документация)
  Prefabs/                Managers/_Managers · UI/UIRoot · Gameplay/Player
  Scenes/                 00_Bootstrap · 01_MainMenu · 02_Game
  ScriptableObjects/
    AudioCues/            (пусто — кидай новые AudioCue сюда)
    Configs/              InputReader
    EventChannels/        OnGameStateChanged · OnScoreChanged · OnGamePaused · OnGameResumed · OnLoadStart · OnLoadComplete
  Scripts/
    Core/                 Singleton · GameBootstrapper · ServiceLocator · ObjectPool
    Editor/Setup/         TemplateSetup · TemplateSceneSetup (инструменты меню Tools/LudumDare)
    Events/               VoidEventChannelSO · IntEventChannelSO · FloatEventChannelSO · StringEventChannelSO · GameStateEventChannelSO
    Gameplay/             (твой игровой код)
    Input/                InputReader (SO) + InputSystem_Actions.inputactions
    Managers/             GameManager · SceneLoader · AudioManager · SaveManager · PauseService · AudioCueSO · GameState
    UI/                   UIScreen · UIManager · MainMenuScreen · SettingsScreen · PauseScreen · HUDScreen · EndGameScreen · LoadingScreen · PauseInputWatcher
    Utilities/            (твои хелперы)
  Settings/               Lit2DSceneTemplate · PostProcessing/GlobalVolumeProfile
```

## Куда что класть

- Новый скрипт геймплея → `Scripts/Gameplay/<подсистема>/`.
- Новый UI-экран → `Scripts/UI/`, префаб (если отдельный) → `Prefabs/UI/Screens/`.
- Новый SO-канал — тип в `Scripts/Events/`, ассет в `ScriptableObjects/EventChannels/`.
- Новый `AudioCueSO` → `ScriptableObjects/AudioCues/`, клипы → `Audio/SFX/` или `Audio/Music/`.
- Редакторные инструменты → `Scripts/Editor/` (и опционально свой подфолдер).
- Художка: спрайты в `Art/Sprites/`, UI-текстуры в `Art/UI/`, шрифты в `Art/Fonts/`, анимации в `Art/Animations/`, VFX в `Art/VFX/`.

## Про префикс `_` в `_Project`

Папка с подчёркиванием всегда всплывает наверх в сортировке. Удобно, когда кроме проекта есть `Plugins/`, `TextMesh Pro/`, `Settings/` — глаз сразу находит основной контент.
